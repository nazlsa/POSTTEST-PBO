# SISTEM PENGELOLAAN STOK BAJU BATIK

**OmahKoe Batik**  
Sistem manajemen stok berbasis OOP (Object Oriented Programming) menggunakan Java.

---

## 📋 Deskripsi Proyek

Proyek ini adalah aplikasi console sederhana untuk mengelola stok produk batik, khususnya **Baju Batik** dan **Aksesori**. Sistem ini mendemonstrasikan konsep OOP meliputi:

- **Abstract Class** (`Produk`)
- **Interface** (`IDiskon`)
- **Inheritance**
- **Polymorphism**
- **Method Overriding & Overloading**

---

## ✨ Fitur

- Menampilkan informasi produk (Baju & Aksesori)
- Perhitungan diskon bertingkat berdasarkan jumlah pembelian
- Manajemen stok (pengurangan stok dengan berbagai overload)
- Format harga yang rapi (Rp xx,xxx.xx)
- Tampilan menu interaktif
- Implementasi Abstract Method dan Interface

---

## 🛠️ Teknologi

- **Bahasa**: Java
- **Paradigma**: Object Oriented Programming (OOP)
- **IDE yang Direkomendasikan**: IntelliJ IDEA / VS Code

---

## 🚀 Cara Menjalankan

1. Buka project di IDE
2. Pastikan file `IDiskon.java` bernama **persis** `IDiskon.java` (huruf I besar)
3. Run file `Main.java`
4. Ikuti menu yang muncul

---

## 📌 Catatan Penting

- Terdapat overload method `kurangiStok()` dan `getInfo()`
- Diskon dihitung melalui interface `IDiskon`
